# PRAYED — Catholic Prayer & Community App

A progressive web app for Catholic families to pray together, share intentions, and build faith communities.

## Features

- **Prayer Circles** — Create and join prayer groups with real-time chat
- **Daily Prayers** — Rosary, Liturgy of the Hours, and custom prayers
- **Family Management** — Add family members and pray together
- **Events & RSVP** — Organize prayer meetings with one-tap RSVP
- **Community Chat** — Real-time messaging within circles and with Holy Cross
- **Share & Invite** — Native share or clipboard link sharing

## Tech Stack

- Single-page HTML app (generated via Python build script)
- Firebase Authentication (Email/Password + Google OAuth)
- Cloud Firestore (real-time database)
- GitHub Pages (hosting)
